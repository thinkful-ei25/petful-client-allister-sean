PETFUL is a web application that helps users easily adopt pets. 
Users are able to see the lastest and greatest pets that are available for adoption.
User are also able to adopt the lastest pets. 
When users adopt a pet it takes the pet out of the queue, so the pet is no longer
avilable for adoption.
